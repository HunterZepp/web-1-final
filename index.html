Portfolio

Master Lock “The Luxury Lock”

The idea behind this project was to take a product that used a lot of plastic and other non-sustainable materials in its packaging and redesign it with sustainability in mind. I chose a Master Lock padlock to redesign as the packaging is made mainly of plastic.

Video Game Console Interface

This project was about either redesigning an existing digital interface or creating a new one from scratch in Figma. The route I chose to go down for this project was to design my own video game console interface.

Ton-Up Boys Booklet

This project is a 12-page booklet about the Ton-Up Boys using a provided set of images and text. To make this project more challenging, I needed to get creative with the placement of the text and images in a way that connects with the theme.

Get Out The Vote Poster

The goal of this project was to design a set of posters for the Get Out the Vote campaign. Each poster is meant to be presented in 3 different formats, an 11x17 poster, a digital poster for screens, and a social media post.
